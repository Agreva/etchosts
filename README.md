# [/etc/hosts Telegram bot](https://t.me/etchosts_bot)
A telegram bot that generates `hosts`

## Setup
* Clone the repository, Then enter into bot directory, and do `npm install`
* Create `.env` file, and fill:

```
BOT_TOKEN=BOT_TOKEN_HERE
```
